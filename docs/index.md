# momo-ai-kit

**Bộ công cụ AI dùng chung cho team MoMo** — skills, agents, commands giúp công việc hàng ngày nhanh hơn.

Không cần biết code để dùng. Chỉ cần Claude Code đã cài trên máy.

## Bắt đầu từ đâu?

- 🎨 **Non-tech (Designer/PM/Marketer)** → [Getting Started 5 phút](getting-started.html)
- 👨‍💻 **Engineer** → [Contributing guide](CONTRIBUTING.html)
- 📚 **Tìm skill có sẵn** → [Catalog](catalog.html)
- 💡 **Đề xuất skill mới** → [Mở issue](https://github.com/jonathankv/momo-ai-kit/issues/new/choose)

## Tại sao cần repo này?

Thay vì mỗi người giữ riêng 1 bộ prompt trong Notion/Google Doc:
- 1 chỗ chung, ai cũng dùng được
- Ai cũng đóng góp được
- Auto-update khi skill improve

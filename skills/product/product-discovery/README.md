# product-discovery

> **Status:** research-preview • **Owner:** @jonathankv • **v0.1.0**

## What this does

AI-powered Product Discovery — giúp PMs làm rigorous discovery: problem framing → user research synthesis → competitive analysis → opportunity sizing → discovery brief.

## Why this exists

Discovery work hay bị "thổi" (wishful sizing, cherry-picked interviews, confirmation bias). Skill này force structure: mọi claim phải có evidence, mọi opportunity phải qua sizing, mọi brief phải có kill criteria.
